Digital Audio Workstations

This page describes how to set up various DAWs so we can receive MIDI and generate sounds.

Logic Pro

https://www.apple.com/uk/logic-pro

  1. Ensure that you've set up MIDI for your OS.
  2. Create a new project with a single software instrument channel (e.g. a Piano).
  3. Under Preferences > MIDI > Inputs check IAC Driver Bus 1.
  4. Under File > Project Settings > Recording check Auto demix by channel if multitrack recording.

Ableton Live

https://www.ableton.com/en/live

  1. Ensure that you've set up MIDI for your OS.
  2. Create a new project with a single software instrument channel (e.g. a Piano).
  3. Under Live > Preferences > MIDI > MIDI Ports set Input: IAC Driver (Bus 1) > Track: ON.

Reason

https://www.propellerheads.com/en/reason

  1. Preferences > Advanced > Bus A: IAC Driver Bus 1
  2. Hardware Interface > Advanced MIDI > Assign instrument to channel

Others

If your app is not listed above, try Googling for APP NAME multi track midi routing.
